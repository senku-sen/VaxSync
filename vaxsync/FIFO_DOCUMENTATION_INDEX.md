## üí° Key Concepts

### FIFO (First-In-First-Out)
- Oldest inventory used first
- Prevents vaccine expiry
- Follows standard practices

### Multiple Records
- Handles split inventory (e.g., TT1: 100 + 100)
- Processes all records in order
- Tracks all affected records

### Detailed Logging
- Console logs show all operations
- Includes batch numbers and quantities
- Helps with debugging

### Error Handling
- Warns if insufficient inventory
- Handles partial deductions
- Never fails silently
